### Day 11 成果总结

**P0 — SVM 模型训练（填补最大缺口）**
- 合成 5544 样本（4 字体 × 3 字号 × 增强），HOG 特征提取，SVM RBF 训练
- 测试准确率 99.52%，模型 11.1 MB
- `tool_svm_predict` 从 `char="?" / confidence=0.0` → 真实预测
- 条件路由真正生效：B (0.88) / 0 (0.87) 跳过 LLM 复核

**P0 — DeepSeek API 三层容错**
- agent/retry.py：asyncio.timeout(30s) + tenacity retry(3次指数退避) + fallback 降级
- 区分可重试（网络/超时/429/503）和不可重试（401/403）异常
- llm_verify_node 集成 safe_llm_call

**P1 — Parallel 并行识别**
- recognize_node 从串行改为 asyncio.gather + ThreadPoolExecutor
- 验证：185x 加速（1130ms → 6.1ms），7 字符结果完全一致

**P1 — Interrupt / Resume 人工确认**
- human_review_node：极低置信度 (< 0.5) 触发人工确认
- graph_state.py 新增 awaiting_human + low_confidence_chars
- graph_agent.py 三级条件路由：human / verify / output
- 文档详解 tRPC-Agent interrupt() API + LangGraph checkpoint

### 防幻觉四道闸门

```
conf ≥ 0.85 → 第1道：直接通过（SVM 高置信度）
0.5 ≤ conf < 0.85 → 第2道：LLM 复核（retry+降级）
                   → 第3道：全部失败降级 SVM 兜底
conf < 0.5 → 第4道：interrupt 人工确认
```

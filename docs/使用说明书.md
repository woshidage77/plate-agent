# PlateAgent 车牌识别智能体系统 — 使用说明书

## 系统概述

PlateAgent 是一个集成了 tRPC-Agent 智能体框架的车牌图像识别系统，提供 Web 端管理后台和 AI 交互界面。

## 技术架构

```
浏览器 (Vue 3 前端, 端口 3001)
    │
    ├── /api/* ──────► 后端 API (端口 8080) ──► SQLite 数据库 (plate_records.db)
    │
    └── /agent-api/* ─► FastAPI Agent (端口 8000) ──► tRPC-Agent + DeepSeek LLM
```

## 启动方式

### 环境要求
- Python 3.11+ (含 pip)
- Node.js 20+
- DeepSeek API Key (已配置在 .env 文件中)

### 启动步骤

1. 启动后端 API 服务（端口 8080）：
   ```
   cd plate-agent
   python mock_api.py
   ```

2. 启动 FastAPI Agent 服务（端口 8000）：
   ```
   cd plate-agent
   python -m uvicorn server.app:app --host 0.0.0.0 --port 8000
   ```

3. 启动前端开发服务器（端口 3001）：
   ```
   cd plate-agent/frontend
   npx vite --port 3001 --host 0.0.0.0
   ```

4. 打开浏览器访问：http://localhost:3001

### Docker 部署（生产环境）
```
cd plate-agent/docker
docker compose up -d
```
将启动 7 个容器：Nginx + FastAPI + Spring Boot + MySQL + Redis + ChromaDB + RabbitMQ

## 页面功能

### 1. 登录页
- 地址：http://localhost:3001/#/login
- 账号：admin / 密码：password

### 2. 监控大屏
- 地址：http://localhost:3001/#/dashboard
- 功能：6 个统计卡片 + ECharts 柱状图/饼图 + 最近识别记录表
- 数据每 10 秒自动刷新

### 3. 识别记录
- 地址：http://localhost:3001/#/records
- 功能：分页记录列表 + 车牌号搜索 + 状态筛选 + 黑名单视图

### 4. AI 实验室（核心页面）
- 地址：http://localhost:3001/#/agent-lab
- 包含两个子面板：

**识别管线：**
- 左侧选择测试图片（30 张车牌样本）
- 点击"开始识别"
- 右侧实时展示 6 节点 Agent 流水线执行进度
- 显示工具调用（tool_call/tool_result）、文本输出（text_delta）
- 最终结果卡片：车牌号、置信度、黑名单状态

**AI 对话：**
- 多轮对话界面
- 输入自然语言指令（如"识别这张车牌"、"查询黑名单"）
- Agent 自动调用工具链完成识别
- 工具调用以标签形式内联展示

## 数据库说明

- 开发环境使用 SQLite，文件位置：plate-agent/plate_records.db
- 表名：plate_records（13 个字段，含车牌号、置信度、黑名单标记、处理耗时等）
- 预置 8 条样本数据，可通过 API 接口增删改查
